HTML5

HTML es un lenguaje de etiquetas significa (HyperText Markup Language)

Y se usa para hacer la estructura de la pagina web


Etiquetas principales

!DOCTYPE

Etiqueta que se usa para indicar que la pagina esta realizada en la version de HTML5

HTML

Dentro de estas etiquetas se debe de poner toda la pagina web


Etiquetas principales dentro de HTML

Head

Head es la cabeza de la pagina web, Aqui agregamos el titulo de la pagina (Pesta単a del navegador),

Enlaces de CSS, Enlaces de JavaScript, Etiquetas Meta y demas enlaces

Body

El Body es el cuerpo de la pagina web, Aqui se agrega todos los componentes de la pagina.

Footer

El Footer es los pies de la pagina web, En el ponemos enlaces a redes sociales, Numeros telefonicos, Direcciones etc.


Contenedores

Nav (Menu)

El NAV o menu es donde ponemos los enlaces de navegacion de la pagina web

Header (Cabezera)

El Header (Cabezera) es donde podemos poner el menu y/o La portada de la pagina web

Aside (A un lado)

Este contenedor suele estar a los lados, Izquierda o Derecha. Y se suele usar para menus alternos o publicidad

Section (Secciones)

Este contenedor se usa para meter secciones de contenido

Article (Articulos)

Los Articulos son peque単os contenedores que se suelen usar dentro de otros contenedores y se usan

para hacer peque単os articulos o cuadros con contenido

DIV (Divisiones) - No recomendable usar

Antiguamente se solia usar pero ya cayo en des-uso, Ya que los dispositivos moviles

no solian reconocer que parte de la pagina era.


Etiquetas HTML5

Etiquetas H

Las etiquetas H van desde el valor 1 hasta el 6.

El valor menor tiene mas importancia que el valor mayor

El valor menor tiene mas tama単o que el valor mayor

h1 h2 h3 h4 h5 h6


P (Parrafos)

Las p son etiquetas que se usan para poner texto de tipo parrafos


A (Enlaces)

La etiqueta a se usa para crear enlaces (Se puede usar en cualquier parte)

La propiedad o atributo que enlaza es el href=""

Llamamos a paginas con su nombre (pagina.html) o a secciones con el # (#seccion1)


IMG (Imagenes)

la etiqueta img se usa para enlazar imagenes

La propiedad o atributo que enlaza es src=""


UL (Agrupar listas)

La etiqueta ul se usa para agrupar li (Listas)


LI (Listas)

Se usa para hacer listas


BUTTON (Boton)

La etiqueta button se usa para hacer botones

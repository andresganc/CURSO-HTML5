HTML5

HTML es un lenguaje de etiquetas significa (HyperText Markup Language)

Y se usa para hacer la estructura de la pagina web


Etiquetas principales

!DOCTYPE

Etiqueta que se usa para indicar que la pagina esta realizada en la version de HTML5

HTML

Dentro de estas etiquetas se debe de poner toda la pagina web


Etiquetas principales dentro de HTML

Head

Head es la cabeza de la pagina web, Aqui agregamos el titulo de la pagina (Pestaña del navegador),

Enlaces de CSS, Enlaces de JavaScript, Etiquetas Meta y demas enlaces

Body

El Body es el cuerpo de la pagina web, Aqui se agrega todos los componentes de la pagina.

Footer

El Footer es los pies de la pagina web, En el ponemos enlaces a redes sociales, Numeros telefonicos, Direcciones etc.


Etiquetas HTML5

Etiquetas H

Las etiquetas H van desde el valor 1 hasta el 6.

El valor menor tiene mas importancia que el valor mayor

El valor menor tiene mas tamaño que el valor mayor

h1 h2 h3 h4 h5 h6
